# Nursery & Seed Management System

This project is developed as part of the CivoraX Internship Program by Civora Nexus Pvt. Ltd.

## Project Overview
The Nursery & Seed Management System is a full stack web application designed to manage inventory, sales, and lifecycle tracking of plants, seeds, and saplings for a commercial nursery.

## Objectives
- Manage nursery inventory with batch-level tracking
- Support role-based access for Staff, Customers, and Admin
- Enable order placement and automatic stock deduction
- Track plant health and lifecycle events
- Generate reports for inventory and sales analysis

## Planned Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MySQL
- Version Control: Git & GitHub
